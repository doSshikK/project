# 🤖 HR Agent — интеллектуальный помощник для HR

**HR Agent** — это AI-помощник для автоматизации HR-процессов, интегрированный с Telegram. Система помогает рекрутерам и HR-специалистам автоматизировать рутинные задачи, позволяя сократить время на подбор персонала и адаптацию сотрудников.

---

## 🚀 Возможности

### 1. 📄 Работа с кандидатами
| Функция | Описание |
|---------|----------|
| **Парсинг резюме** | Автоматическое извлечение ФИО, контактов, навыков, опыта и должности из PDF/DOCX |
| **Умный поиск** | Поиск кандидатов по любым навыкам (Python, CSS, React, аналитика и т.д.) с фильтрацией по опыту. Поддерживает любые профессии: IT, производство, строительство, логистика, офис, продажи, маркетинг, финансы, HR |
| **Управление базой** | Добавление, редактирование, удаление, восстановление из корзины |
| **Архивация** | Отправка кандидатов в архив с указанием причины (найм/отказ) |
| **Экспорт в PDF** | Сохранение карточки кандидата с отправкой в чат |
| **Экспорт в Excel** | Выгрузка всех кандидатов в Excel-файл для анализа |

**Алгоритм поиска:**
- Извлекает ключевые слова из запроса пользователя
- Фильтрует кандидатов по наличию искомых навыков
- Сортирует по количеству совпадений
- Поддерживает фильтрацию по опыту работы

### 2. 💼 Работа с вакансиями
| Функция | Описание |
|---------|----------|
| **Создание вакансий** | Название, уровень (junior/middle/senior), навыки, опыт, описание |

| **Архивация** | Отправка вакансий в архив и восстановление |
| **Управление статусом** | Активные / архивные вакансии |
| **Экспорт в PDF** | Сохранение карточки вакансии |

## 🛠 Технологии

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **LLM** | OpenRouter API (GPT-4o-mini / Qwen / Gemma) | Генерация ответов и function calling |
| **Messenger** | python-telegram-bot v20+ | Асинхронный Telegram бот |
| **Database** | PostgreSQL + psycopg2 | Хранение кандидатов, вакансий, опросов |
| **Config** | Pydantic v2 + python-dotenv | Настройки из .env |
| **PDF** | ReportLab | Экспорт в PDF с поддержкой кириллицы |
| **Excel** | openpyxl | Экспорт в Excel |
| **Charts** | matplotlib | Генерация графиков опросов |
| **PDF Parsing** | pdfplumber, PyMuPDF, python-docx | Парсинг резюме |
| **Fuzzy Search** | rapidfuzz | Нечёткое сравнение навыков |
| **Logging** | loguru | Логирование с ротацией |
| **Deployment** | Docker, Docker Compose | Контейнеризация |

### Паттерны проектирования

| Паттерн | Где используется |
|---------|------------------|
| **Facade** | `HRAgentFacade` — единая точка входа для бота |
| **Dependency Injection** | `HRAgent(llm_client, model)` — возможность подмены зависимостей |
| **Repository** | `candidate_db`, `jobs_db`, `survey_db`, `interview_db` — изоляция SQL |
| **Singleton** | `settings`, `logger`, пул соединений PostgreSQL |
| **Template Method** | `test_generator` — LLM генерация с fallback на шаблоны |
| **Strategy** | `job_matcher` — алгоритмы скоринга (навыки, опыт, образование) |

# 📦 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/your-username/hr-agent.git
cd hr-agent
```

### 2. Создание виртуального окружения

```
# macOS/Linux
python3 -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

```
pip install -r requirements.txt
```

### 4. Установка PostgreSQL

macOS:
```
brew install postgresql@15
brew services start postgresql@15
```

Ubuntu/Debian:
```
sudo apt update
sudo apt install postgresql-15
sudo systemctl start postgresql
```

Windows:

Скачайте и установите PostgreSQL с [официального сайта](https://www.postgresql.org/download/windows/)

Создание базы данных:
```
sudo -u postgres psql
```
В интерактивной консоли PostgreSQL выполните:

```
CREATE DATABASE hr_agent;
CREATE USER hr_agent WITH PASSWORD 'your_strong_password';
GRANT ALL PRIVILEGES ON DATABASE hr_agent TO hr_agent;
\q
```

### 5. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```bash
# ============================================================
# ОБЯЗАТЕЛЬНЫЕ НАСТРОЙКИ
# ============================================================

# OpenRouter API (для LLM) — получить на https://openrouter.ai
OPENROUTER_API_KEY=sk-or-v1-xxxx...

# Telegram Bot — получить у @BotFather
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

# PostgreSQL
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_DB=hr_agent
POSTGRES_USER=hr_agent
POSTGRES_PASSWORD=your_strong_password

# HR Telegram ID (узнать у @userinfobot)
HR_TELEGRAM_ID=123456789

# ============================================================
# ОПЦИОНАЛЬНЫЕ НАСТРОЙКИ
# ============================================================

# Дополнительные HR ID (через запятую)
HR_IDS=

# Настройки LLM
DEFAULT_MODEL=openai/gpt-4o-mini
LLM_TIMEOUT=180
LLM_TEMPERATURE=0.5

# Настройки поиска (сумма весов должна быть = 1.0)
DEFAULT_MATCH_SKILLS_WEIGHT=0.5
DEFAULT_MATCH_EXP_WEIGHT=0.3
DEFAULT_MATCH_POSITION_WEIGHT=0.2

# Логирование
LOG_LEVEL=INFO

# SMTP для отправки писем (опционально)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=
SMTP_PASSWORD=

# IMAP для приёма резюме по почте (опционально)
EMAIL_HOST=imap.gmail.com
EMAIL_PORT=993
EMAIL_USER=
EMAIL_PASSWORD=
```

### 6. Получение Telegram Bot Token

1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Придумайте имя и username бота
4. Скопируйте полученный токен в `.env` (переменная TELEGRAM_BOT_TOKEN)


### 7. Получение OpenRouter API Key

1. Зарегистрируйтесь на [OpenRouter](https://openrouter.ai)
2. Перейдите в раздел API Keys
3. Создайте новый ключ (начинается с `sk-or-v1-`)
4. Скопируйте ключ в `.env`(переменная OPENROUTER_API_KEY)

### 8. Получение HR Telegram ID

Напишите @userinfobot в Telegram

Нажмите /start

Скопируйте ваш ID в .env (переменная HR_TELEGRAM_ID)

### 9. Запуск через Docker (рекомендуется)

```
# Создать .env файл (см. шаг 5)
cp .env.example .env
# Отредактировать .env, указав свои ключи

# Запустить контейнеры
docker-compose up -d

# Посмотреть логи
docker-compose logs -f bot

# Остановить
docker-compose down
```

### ▶️ Запуск

```
python main.py
```

## Формула скоринга кандидатов

**Итоговая оценка совпадения кандидата с вакансией:**

$$
\text{Score} = w_{\text{skills}} \times \text{SkillsMatch} + w_{\text{exp}} \times \text{ExperienceMatch} + w_{\text{edu}} \times \text{EducationMatch}
$$

### Компоненты

| Компонент | Формула | Вес (по умолчанию) |
|-----------|---------|---------------------|
| **SkillsMatch** | $\frac{\text{matched\_skills}}{\text{required\_skills}}$ (нормализовано, с бонусом за редкие навыки) | $w_{\text{skills}} = 0.5$ |
| **ExperienceMatch** | $\min\left(\frac{\text{candidate\_exp}}{\text{required\_exp}}, 1.0\right)$ | $w_{\text{exp}} = 0.3$ |
| **EducationMatch** | $1.0$ (базовое, можно расширить) | $w_{\text{edu}} = 0.2$ |
