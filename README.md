# 🤖 HR Agent — интеллектуальный помощник для HR

**HR Agent** — это AI-помощник для автоматизации HR-процессов, интегрированный с Telegram. Система помогает рекрутерам и HR-специалистам автоматизировать рутинные задачи, позволяя сократить время на подбор персонала и адаптацию сотрудников.

---

## 🚀 Возможности

### 1. 📄 Работа с кандидатами
| Функция | Описание |
|---------|----------|
| **Парсинг резюме** | Автоматическое извлечение ФИО, контактов, навыков, опыта и должности из PDF/DOCX |
| **Умный поиск** | Поиск кандидатов по любым навыкам (Python, CSS, React, аналитика и т.д.) с фильтрацией по опыту. Поддерживает любые профессии: IT, производство, строительство, логистика, офис, продажи, маркетинг, финансы, HR |
| **Управление базой** | Добавление, редактирование, удаление, восстановление из корзины |
| **Архивация** | Отправка кандидатов в архив с указанием причины (найм/отказ) |
| **Экспорт в PDF** | Сохранение карточки кандидата с отправкой в чат |
| **Экспорт в Excel** | Выгрузка всех кандидатов в Excel-файл для анализа |

**Алгоритм поиска:**
- Извлекает ключевые слова из запроса пользователя
- Фильтрует кандидатов по наличию искомых навыков
- Сортирует по количеству совпадений
- Поддерживает фильтрацию по опыту работы

### 2. 💼 Работа с вакансиями
| Функция | Описание |
|---------|----------|
| **Создание вакансий** | Название, уровень (junior/middle/senior), навыки, опыт, описание |
| **Matching кандидатов** | Автоматический подбор кандидатов под вакансию с расчётом % совпадения |
| **Архивация** | Отправка вакансий в архив и восстановление |
| **Управление статусом** | Активные / архивные вакансии |
| **Экспорт в PDF** | Сохранение карточки вакансии |

### 3. 📝 Генерация тестовых заданий
| Функция | Описание |
|---------|----------|
| **Универсальные направления** | IT, Производство, Строительство, Логистика, Офис, Продажи, Маркетинг, Финансы, HR (всего 15+) |
| **3 уровня сложности** | Junior, Middle, Senior |
| **Уникализация** | LLM-генерация с высокой температурой + уникальные темы |
| **Fallback механизм** | Шаблоны при недоступности LLM |
| **Редактирование** | Изменение направления, уровня, стека технологий |
| **Экспорт в PDF** | Сохранение тестового задания с отправкой в чат |

### 4. 📋 Генерация планов онбординга
| Параметр | Описание |
|----------|----------|
| **9 отделов** | Разработка, Аналитика, Менеджмент, Маркетинг, Продажи, HR, Бухгалтерия, Юридический, Дизайн |
| **Уровни сложности** | Junior / Middle / Senior (для отдела разработки) |
| **Автоопределение** | Отдел и уровень определяются автоматически из текста должности |
| **Чек-лист задач** | Персонализированные задачи на 1-5 дней с дедлайнами |
| **План встреч** | Расписание знакомств с ключевыми сотрудниками |
| **Интерактивный режим** | Пошаговое прохождение, отметка о выполнении, напоминания |
| **Экспорт в PDF** | Сохранение плана с отправкой в чат |
| **Редактирование** | Изменение имени, должности, даты начала |

### 5. 📊 Опросы (NPS и Pulse)
| Функция | Описание |
|---------|----------|
| **NPS опросы** | Оценка лояльности (0-10), расчёт NPS Score, распределение промоутеров/нейтралов/критиков |
| **Pulse опросы** | Ежедневный/еженедельный мониторинг удовлетворённости и энергии сотрудников (1-5) |
| **Графики** | Столбчатая диаграмма NPS, линейный график динамики Pulse |
| **Ссылка для сотрудников** | `/start survey_{id}` — анонимное прохождение |
| **Аналитика** | Средние оценки, распределение ответов, текстовые отзывы |

### 6. 📅 Управление собеседованиями
| Функция | Описание |
|---------|----------|
| **Слоты для собеседований** | Добавление, просмотр, удаление слотов |
| **Приглашения** | Отправка приглашений кандидатам в Telegram или по email |
| **Бронирование** | Кандидат выбирает удобную дату и время |
| **Напоминания** | Автоматические напоминания о собеседовании |
| **Офферы** | Отправка предложений о работе с выбором даты выхода |

### 7. 📋 Интерактивный онбординг
| Функция | Описание |
|---------|----------|
| **Пошаговое прохождение** | Кандидат проходит адаптацию шаг за шагом |
| **Напоминания** | Автоматические напоминания о дедлайнах задач |
| **Прогресс** | Визуальный прогресс-бар выполнения задач |
| **Помощь** | Подробные инструкции по каждой задаче |
| **Встречи** | План встреч с командой |

### 8. 🤖 Telegram бот
| Функция | Описание |
|---------|----------|
| **Интуитивные команды** | `найди Python разработчика`, `создай тест для frontend junior` |
| **Кнопки-меню** | Удобная навигация по разделам |
| **Пошаговые сценарии** | Добавление кандидата/вакансии с валидацией |
| **Пагинация** | Кнопки "Далее/Назад" для длинных списков |
| **Подтверждение действий** | Удаление требует подтверждения |
| **Экспорт в PDF** | Любой отчёт можно сохранить и отправить в чат |

## 🛠 Технологии

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **LLM** | OpenRouter API (GPT-4o-mini / Qwen / Gemma) | Генерация ответов и function calling |
| **Messenger** | python-telegram-bot v20+ | Асинхронный Telegram бот |
| **Database** | PostgreSQL + psycopg2 | Хранение кандидатов, вакансий, опросов |
| **Config** | Pydantic v2 + python-dotenv | Настройки из .env |
| **PDF** | ReportLab | Экспорт в PDF с поддержкой кириллицы |
| **Excel** | openpyxl | Экспорт в Excel |
| **Charts** | matplotlib | Генерация графиков опросов |
| **PDF Parsing** | pdfplumber, PyMuPDF, python-docx | Парсинг резюме |
| **Fuzzy Search** | rapidfuzz | Нечёткое сравнение навыков |
| **Logging** | loguru | Логирование с ротацией |
| **Deployment** | Docker, Docker Compose | Контейнеризация |
